Esta es la estructura básica de un documento HTML: ``` Document ``` : Esto nos dice que es un documento HTML5. : Contiene todo el contenido de la página. : Contenido principal (textos, imagenes, tablas, etc). : Aqui ponemos datos visibles para nosotros los que creamos el codigo, aparte tambien ponemos el titulo de la pagina y diferentes tipos de enlaces. Vamos a hablar un poco mas del head porque es importante, aqui hemos de poner metadatos importantes: - : Define la codificación de caracteres (UTF-8 es la más común). - : Hace que la página sea responsiva en dispositivos móviles. - : Proporciona una breve descripción de la página para motores de búsqueda. - : Palabras clave relacionadas con el contenido de la página. - : Nombre del autor de la página. - : Titulo de la pestaña del navegador. - <link>: Enlaces a hojas de estilo (CSS) o scripts (JavaScript). #### Encabezados En html los encabezados se hacen de la siguiente manera, es decir con la etiqueta h1, h2, h3, h4, h5 y h6. La diferencia entre ellos es el tamaño, siendo el 1 el mas grande y el 6 el mas pequeño. ``` <h1>Encabezado 1</h1> <h2>Encabezado 2</h2> <h3>Encabezado 3</h3> <h4>Encabezado 4</h4> <h5>Encabezado 5</h5> <h6>Encabezado 6</h6> ``` #### Parrafos y texto Para poner texto utilizamos la etiqueta p, para poner texto en negrita utilizamos la etiqueta strong, para poner texto en cursiva utilizamos la etiqueta em. ``` <p>Este es un parrafo</p> <strong>Texto en negrita</strong> <em>Texto en cursiva</em> ``` Y la forma correcta de ponerlo es la siguiente: ``` <p><strong>Este es un parrafo</strong></p> ``` #### Enlaces Para poner un enlace utilizamos la etiqueta a, y para poner una imagen utilizamos la etiqueta img. <a href="https://www.google.com">Texto del enlace</a> <img src="/img/descarga.jpeg" alt="Texto alternativo" width="200" height="100"> href: URL de destino, se le pueden añadir atributos como target="_blank" para abrir en una nueva pestaña o window.open("url") para abrir en una ventana emergente. img src: URL de la imagen, alt: texto alternativo, width: ancho de la imagen, height: alto de la imagen. #### Listas Para poner una lista utilizamos la etiqueta ul (lista desordenada) o la etiqueta ol (lista ordenada), y para poner un elemento de la lista utilizamos la etiqueta li. <!-- Lista Ordenada (numerada) --> <ol> <li>Elemento 1</li> <li>Elemento 2</li> </ol> <!-- Lista Desordenada (viñetas) --> <ul> <li>Elemento A</li> <li>Elemento B</li> </ul> #### Tablas Para poner una tabla utilizamos la etiqueta table, y para poner una fila utilizamos la etiqueta tr, y para poner una celda utilizamos la etiqueta td. <table> <tr> <th>Encabezado 1</th> <th>Encabezado 2</th> </tr> <tr> <td>Dato 1</td> <td>Dato 2</td> </tr> </table> #### Elementos de estructura Hay muchos elementos de estructura, aqui vemos algunos de ellos: <header> Contenido que se encuentra en la parte superior de la página, puede contener encabezados, barras de navegación, o logos. </header> <main> Contenido principal de la página. </main> <nav> Barra de navegación que puede contener enlaces a otras páginas o secciones del mismo sitio. </nav> <footer> Contenido que se encuentra en la parte inferior de la página, puede contener copyright, información sobre el sitio, o redes sociales. </footer> <article> Contenido independiente, puede ser un artículo, una noticia, un comentario, etc. </article> <section> Sección de contenido relacionado, puede contener varios artículos o párrafos. </section> <aside> Contenido que se encuentra a la derecha del contenido principal, puede ser un sidebar, una lista de enlaces, o información relacionada con el artículo. </aside> <figure> Contenido que se relaciona con un solo elemento visual, puede ser una imagen, un video, un audio, o cualquier otro elemento multimedia. </figure> #### Formularios Para poner formularios utilizamos la etiqueta form, y para poner un campo de texto utilizamos la etiqueta input, dentro del input podemos poner el tipo de campo que queremos, como por ejemplo text, password, email, etc. Para poner un botón utilizamos la etiqueta button.
